## サイト概要
### サイトテーマ
タグやワードでタイムラインを絞り込める　SNSサイト
​
### テーマを選んだ理由
好きなことを人と共有したい　好きなことを優先的に知りたいそう思ったからです
​
### ターゲットユーザ
推しを布教したかったり　他の人と共感したい人
​
### 主な利用シーン
- 推しを布教するとき
- 集中的にその情報を調べたい時
